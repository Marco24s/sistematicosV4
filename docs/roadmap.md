# Roadmap sugerido

1. Migraciones Alembic y seeds controlados para catalogos iniciales.
2. Autenticacion, roles y permisos por funcion tecnica.
3. Documentos tecnicos con hash, versionado y trazabilidad.
4. Instalacion/remocion formal de componentes con eventos e historial.
5. Ordenes de trabajo con firmas, control de calidad y liberacion.
6. Control de habilitaciones del personal tecnico.
7. Logistica interna: almacenes, cuarentena, reservas y despacho.
8. Outbox transaccional para publicar eventos sin perdida.
9. Dashboard operacional de aeronavegabilidad y vencimientos.
