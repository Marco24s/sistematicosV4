# API inicial

Base URL: `/api/v1`

## Configuracion

- `POST /configuration/aircraft-models`
- `GET /configuration/aircraft-models`
- `POST /configuration/engine-models`
- `GET /configuration/engine-models`
- `POST /configuration/component-types`
- `GET /configuration/component-types`

## Activos aeronauticos

- `POST /assets/aircraft`
- `GET /assets/aircraft`
- `POST /assets/components`
- `GET /assets/components`
- `GET /assets/aircraft/{aircraft_id}/airworthiness`
